# Ruleta para B4A

Proyecto base creado para abrirse y ejecutarse con el IDE B4A (Basic for Android).

## Archivos principales


- `Ruleta.b4a`: proyecto principal de B4A con la cabecera de metadatos requerida por el IDE y una Activity simple.
- `Starter.bas`: servicio Starter estándar usado por proyectos B4A recientes.
- `Ruleta.b4a`: proyecto principal de B4A con una Activity simple.
- `.gitkeep`: marcador original del repositorio.

## Cómo abrirlo en B4A

1. Abre B4A.
2. Selecciona **File > Open Source**.
3. Elige el archivo `Ruleta.b4a`.
4. Configura el SDK de Android si B4A lo solicita.
5. Ejecuta el proyecto en un emulador o dispositivo Android.

## Funcionalidad inicial

La app muestra una pantalla simple de ruleta con:

- título `Ruleta`;
- resultado inicial `Presiona Girar`;
- botón `Girar`;
- selección aleatoria de un número de ruleta europea entre 0 y 36;
- color del resultado: verde para 0, rojo o negro para los demás números.
